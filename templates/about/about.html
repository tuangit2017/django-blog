Python and Data Science

Project idea will be perform analysis of movies released in the cinema between 2000 and 2019 where we use an API to interface with IMDB and get practice with plotting and data visualizations using modules such as matplotlib, pandas, and seaborn. We also build an website using Django framework to display the result as well as information of the projects

Introduction

The Internet Movie Database (IMDb) is a website that serves as an online database of world cinema. This website contains a large number of public data on films such as the title of the film, the year of release of the film, the genre of the film, the audience, the rating of critics, the duration of the film, the summary of the film, actors, directors and much more. Faced with the large amount of data available on this site, so, it would be interesting to analyze the movies data on the IMDb website between the year 2000 and the year 2019.
